# Lab5 — Веб-чат с ИИ-ассистентом

## Описание проекта
Это веб-проект, содержащий HTML, CSS и JavaScript. Главная страница `index.html` отображает содержимое сайта, а дополнительные файлы стилей и скриптов подключены отдельно.  

Проект включает интерактивный **ИИ-чат**, который помогает пользователю получать советы по готовке и рецепты.
